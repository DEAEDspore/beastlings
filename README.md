# Beastlings Game

Мини-игра Beastlings, созданная для Telegram. 
- HTML-файлы находятся в папке `docs/`.
- Стили и скрипты в папке `static/`.
- Код Telegram-бота находится в папке `bot/`.

## Как запустить
1. Клонировать репозиторий.
2. Убедиться, что сайт доступен через GitHub Pages.
3. Запустить `bot.py` для активации бота.
